# HomeKonnectz

A property marketplace for **Ondo and Osun States**, Nigeria, connecting house seekers with verified landlords and agents.

## Features

- **Role-based access**: Free vs Premium users
- **Seller limits**: 
  - Free: 2 property listings
  - Premium: Up to 10 listings with advanced features (video, map, contact)
- **Property verification** during sign-up
- **Interactive map** (premium feature)
- **Responsive design** for all devices

## How to Run

1. Clone this repository
2. Open `index.html` in any modern web browser
3. No server or build tools required!

## Tech Stack

- HTML5
- CSS3 (Flexbox, Grid)
- Vanilla JavaScript
- Font Awesome Icons
- Google Fonts
